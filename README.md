1. ১/ var এটা শুধুমাত্র ফাংশনের ভিতরে সীমাবদ্ধ, ফাংশনের বাইরে হলে global হয়।
    ২/ let একই block-এ আবার declare করা যাবে না।
    ৩/ const Re-declaration করা যায় না।

2. ১/ spread operator দিয়ে object এর elements বা properties কে বিখণ্ডিত (spread) করতে ব্যবহার করা হয়।
    ২/ spread operator  দিয়ে object এর properties কপি করা যায়।
    ৩/Array এর ক্ষেত্রে spread operator দিয়ে array এর সব elements আলাদা করে অন্য array বা function এ ব্যবহার করা যায়।

3. ১/map()এর প্রতিটি element-এ কোন operation/processing করে নতুন array বানায়।
    2/filter() Array-এর element-গুলো কোনো condition মেনে যায় কি না যাচাই করে নতুন array বানায়।
    3/forEach() Array-এর প্রতিটি element-এর জন্য একটা function execute করে।

4. Arrow Function হল একটি সংক্ষিপ্ত (shorter) syntax function লিখার জন্য। এটা ES6  থেকে এসেছে।

5. Template literal backticks `` দিয়ে লেখা হয়, এর মধ্যে ${} ব্যবহার করে variables বা expressions embed করা যায়।
